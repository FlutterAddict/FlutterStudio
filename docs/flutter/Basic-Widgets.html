Basic Widgets

Learn about basic, most commonly used Flutter widgets.

Widgets, Widgets Everywhere

In Flutter, you build the User Interfaces using Widgets.

Almost everything in a Flutter app is a Widget.

We have Widgets for layouts, for animations and even the app itself is a Widget.

Basic Widgets

Flutter comes with a suite of powerful basic widgets, of which the following are very commonly used.

Text() Widget

This widget lets you create a run of styled text within your application.

import 'package:flutter/material.dart';

void main() {
  runApp(
    Center(
      child: Text('Hello World',
        textDirection: TextDirection.ltr
      ),
    ),
  );
}

Container() Widget

This Widget creates a rectangular visual element that can be decorated with a BoxDecoration, such as background, a border or a shadow.

A Container() Widget can also have margins, padding and constraints applied to its size.

In addition, it can be transformed in three dimensional space using a matrix.

Example

import 'package:flutter/material.dart';

void main() {
  runApp(
    Container(
      padding: const EdgeInsets.symmetric(horizontal: 8.0),
      decoration: BoxDecoration(color: Colors.blue[500]),
      child: Text('Hello from a Container!', 
        textDirection: TextDirection.ltr
      ),
    ),
  );
}

Row() and Column() Widgets

These flex widgets let you create flexible layouts in both horizontal (Row() Widget) and verical (Column() Widget) directions. It is a design based on the web's flexbox layout model.

Example

import 'package:flutter/material.dart';

void main() {
  runApp(
    Column(
      children: <Widget>[
        Row(
          children: <Widget>[
            Text('A', textDirection: TextDirection.ltr),
            Text('B', textDirection: TextDirection.ltr),
          ],
        ),
        Row(
          children: <Widget>[
            Text('C', textDirection: TextDirection.ltr),
            Text('D', textDirection: TextDirection.ltr),
          ],
        ),
      ],
    ),
  );
}

Stack() Widget

This widget lets you create stack widgets on top of each other in paint order.

You can then use the Positioned() widget on a children of a Stack() widget to position them relative to the top, right, bottom or left edge of the stack.

Stacks are based on the web's absolute positioning layout model.

Example

Lets create an app that will show a 300x300px red rectangle with Text widget within it (or on top of it), absolutely positioned on (x: 100, y: 100), so (top: 100, left: 100).

import 'package:flutter/material.dart';

void main() {
  runApp(
    Stack(
      width: 300.0,
      height: 300.0,
      color: Colors.red[300],
      children: [
        Positioned(
          child: Text('Hello', textDirection: TextDirection.ltr),
          left: 100.0,
          top: 100.0
        ),
      ],
    ),
  );
}
