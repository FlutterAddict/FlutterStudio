User Interface Basics

Widgets, Widgets Everywhere

In Flutter, we build user interfaces using Widgets.

Everything in Flutter app is a widget.

You have widgets for animations, for building layouts, and even the app itself is a widget.

Hello World

Let's write an app that will only show "Hello World" in the middle of the screen:

import 'package:flutter/material.dart'; // import Flutter to our Dart program

// main, as you should already know is a special, top-level, required function
void main() {
  runApp(
    Center(
      child: Text(
        'Hello World',
        textDirection: TextDirection.ltr,
      ),
    ),
  );
}

Let's break it down.

  1. import 'package:flutter/material.dart'; is basically how we import the Flutter framework into our Dart program.

  2. void main() { ... } as you may already from Dart guide, is the entry point of any Dart program.

  3. runApp(...) makes the given widget the root of the widget tree and forces it to cover the screen.

  4. Center(...) is a simple widget that centers its child within itself.

  5. Text(...) is a widget that displays the text to the screen. The textDirection must be specified when we don't use MaterialApp (we'll get into this later).

Summary

And that's it. In summary - all we do is:

  • import the Flutter's material.dart package

  • create the void main() method

  • call the runApp(RootWidget) method provided by Flutter's material.dart package

  • and pass the widget we want to be the root of our application to it, which is in this case the Center() widget.

  • We pass the Text() widget to the Center() widget as its child.
