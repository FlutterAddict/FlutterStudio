Hello World App

So you know already know Dart and have installed and setup Flutter on your machine.

Let's create a very simple Hello World app with Flutter.

The app will only show a 'Hello World' text in the center of the screen.

1. Create New Project

This one is very simple. Just open VS Code, open the terminal by pressing ctrl + ` and run the command:

flutter create hello_world

This will initialize new Flutter project and make a new directory named hello_world.

2. Open lib/main.dart

The root source file of your app is in lib/main.dart. Open this file and delete its contents.

3. Import Flutter Package

Now lets import the Flutter package to our Dart program:

// main.dart
import 'package:flutter/material.dart';

4. Create Main Function

As you should already know from our Dart Basics, all Dart programs start from the main() function.

It is a required, special, top-level function, where the program starts.

// main.dart
import 'package:flutter/material.dart';

void main() {
  
}

5. Call the runApp() Function

The runApp() is a function provided by Flutter. We imported it with many other in the step 3.

It makes the widget we give to it the root of widget tree and forces it to cover the screen.

void main() {
  runApp(
    // We'll pass the root widget in next step
  );
}

6. Pass our Root Widget

In this app, the root widget will be a widget named Center(), which is a simple widget that centers its child within itself.

void main() {
  runApp(
    Center(

    ),
  );
}

7. Pass the Text Widget

It's pretty straightforward:

runApp(
  Center(
    child: Text('Hello World',
      textDirection: TextDirection.ltr,
    ),
  ),
),

Yeah, the textDirection must be specified if it's not a descedant of the MaterialApp() widget.

Later we'll learn how to use MaterialApp() widget and we'll not have to specify the textDirection every time.

Congratulations

Congrats! You made it. Here's the complete source code of our Flutter Hello World app:

import 'package:flutter/material.dart';

void main() {
  runApp(
    Center(
      child: Text('Hello World',
        textDirection: TextDirection.ltr,
      ),
    ),
  );
}
