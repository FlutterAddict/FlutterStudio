Custom Widgets

Learn what are Stateless and Stateful widgets and how to create your own, custom Flutter widgets.

StatelessWidget vs StatefulWidget

There are 2 kinds of Flutter Widgets - StatefulWidgets and StatelessWidgets.

If a Widget manages any state - it is a StatefulWidget. If not - it's StatelessWidget.

Widget's build() Function

A widget's main job is to implement a build() function, which describes a widget in terms of other, lower-level widgets.

The framework builds those widgets in turn until the process bottoms out in widgets that represent the underlying RenderObject, which computes and describes the geometry of the Widget.
