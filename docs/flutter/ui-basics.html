User Interface Basics

Widgets, Widgets Everywhere

In Flutter, we build user interfaces using Widgets.

Everything in Flutter app is a widget.

You have widgets for animations, for building layouts, and even the app itself is a widget.

Flutter is Reactive

Widgets describe what their view should look like given their current configuration and state.

When a widget's state changes, the widget rebuilds its description and the framework diffs against previous description in order to determine minimal changes needed in the underlying render tree to transition from one state to next.

That makes Flutter Reactive.

Hello World

Let's write an app that will only show "Hello World" in the middle of the screen:

import 'package:flutter/material.dart';

void main() {
  runApp(
    Center(
      child: Text(
        'Hello World',
        textDirection: TextDirection.ltr,
      ),
    ),
  );
}
