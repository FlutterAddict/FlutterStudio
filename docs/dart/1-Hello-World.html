Hello World Program

Let's write the most basic Dart program that will only output a 'Hello World' text.

1. Define Main Function

void main() {
    
}

main() is a required, special, top-level function which is an entry point of the program.

The void means that the function doesn't return anything.

2. Output Hello World

Dart gives us a handy way to display output with print() function.

It takes a string and simply prints it.

void main() {
  print('Hello World');
}

The 'Hello World' in this program is called a string literal.

In Dart, string literals can be created with either single quotes or double quotes, so the following code will be correct as well:

void main() {
  print("Hello World!");
}
