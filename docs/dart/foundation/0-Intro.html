Dart Introduction

Flutter is Dart

Flutter is not a programming language itself.

It's a framework written as an additional layer on top of Dart.

Why Learn Dart

So if Flutter is made of 100% Dart, it seems like good idea to know Dart before going for Flutter, isn't it?

Learning Dart first will allow you to fully understand the Flutter code you write.

You will much more comfortable building apps and coding complex logic when it's needed.

Play with Dart

As you learn, you can play with Dart from your browser, without installing anything thanks to DartPad.

It is an open-source tool that lets you run Dart programs in any modern browser.
