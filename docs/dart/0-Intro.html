Dart Introduction

Flutter is Dart

Flutter is not a programming language itself.

It's a framework, an SDK, an additional layer written on top of Dart programming language.

Why Learn Dart

Learning Dart first will allow you to fully understand the Flutter code you write.

You will much more comfortable building apps and coding complex logic when it's needed.

Play with Dart

You can play with Dart from your browser, without installing anything thanks to DartPad.

It is an open-source tool that lets you run Dart programs in any modern browser.
