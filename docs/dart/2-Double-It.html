1. Define Your Function

You define your custom Dart functions by this syntax:

returnType functionName(param1type param1name, param2type param2name ,...) {
  
}

So let's apply it to our printDoubled() function.

The function won't return anything. It will just print number to the screen. Therefore the return type of this function is void:

void

Next, our function name.

void printDoubled

Next are parameters. This function will take one parameter, a number.

In Dart, a type for integers, whole numbers without floating point is int.

The name of the parameter is up to you, but be careful to not use a Dart reserved keyword as its name.

For example you can't call it num because it's a Dart special, reserved keyword.

We will just call it n.

void printDoubled(int n) {

}
